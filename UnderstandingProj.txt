-To Run/Set up the project:
*Make sure you have npm installed and all the libraries react needs
*To initiate the project, run this command in the project terminal: npx react-native start
*Then you will be prompted some options and just follow them

-How each section of the code works:

*First section of the code is importing all the necessary components and utilities for the file
*Second section is the JSX syntax used for rendering components in the file, visual part of the app
*Third section is JS, the logic components of how stuff behaves and responds
*Fourth section is the CSS which is where we define how each thing in the app looks
